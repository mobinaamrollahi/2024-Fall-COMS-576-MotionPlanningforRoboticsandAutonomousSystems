Put the following files in the same folder or in your PYTHONPATH
* main.py
* geometry.py
* obstacle.py
* planning.py
* queue.py
* draw_cspace.py

For Problem 1 run:

$ python main.py --alg rrt

For Problem 2, run

$ python main.py --alg prm

Please make sure to run the code with python=3.8.5, otherwise there must be some mismatches within the outcome. In addition, you have to install the networkx library; 
you may want to run the following command: pip install networkx.
